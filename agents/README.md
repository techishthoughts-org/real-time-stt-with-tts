# RealtimeSTT Python Backend

This is the Python backend component of the RealtimeSTT system, providing real-time speech-to-text processing with AI integration.

## Features

- Real-time speech recognition using RealtimeSTT
- AI-powered responses with RAG capabilities
- WebSocket support for real-time communication
- Text-to-speech output
- Audio device management
- FastAPI REST API

## Installation

```bash
uv sync
```

## Usage

```bash
uv run python -m src.realtime_stt.main
```

The server will start on http://localhost:8000
